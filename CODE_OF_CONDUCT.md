# Code of Conduct

This project follows the principles of the Contributor Covenant Code of Conduct.

We are committed to providing a welcoming, respectful, and harassment-free environment for everyone.

Harassment, hostility, or disrespect toward contributors — or toward the memory of the original author — will not be tolerated.

---

## Expected Behavior

- Be respectful and professional
- Assume good intent
- Keep discussions constructive and on-topic
- Respect differing levels of experience

---

## Unacceptable Behavior

- Harassment or personal attacks
- Disrespectful or inflammatory language
- Disruptive behavior in issues or pull requests

---

## Enforcement

Project maintainers may remove content or restrict participation that violates this code of conduct.

For concerns, please open a GitHub issue or contact the repository maintainers.
